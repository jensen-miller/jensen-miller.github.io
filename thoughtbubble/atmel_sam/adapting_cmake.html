Atmel SAM | Adapting to a CMake build system

Dependencies

When developing for an Atmel SAM board (i.e. SparkFun SAMD51 ThingPlus, Adafruit Feather M0 Express), the base requirements include:

  • GCC ARM Embedded Toolchain
  • An Atmel DFP pack suited for your board
Between these two dependencies, a program will need to know:
  • Include directory of CMSIS
  • Include directory of DFP
  • Linker Script location
